# sArena Reloaded DIY

基于 sArena Reloaded v2.4.3 的二次开发版本。

**DIY: DK-姜世离(燃烧之刃)**

## 更新记录

- **宠物框体** — 新增竞技场对手宠物生命条，支持位置/大小/颜色/文字等配置
- **右键焦点可配置** — 可关闭右键设焦点，兼容 Clique 等点击施法插件
- **DR 锚定模式** — 递减图标可选仅框体、仅血条、或双显
- **血条饰品镜像** — 可在血条旁显示饰品 CD
- **DR 免疫发光** — 递减免疫时图标脉冲发光
- **自身/队友 DR** — 基于 C_LossOfControl 追踪己方递减状态（仅竞技场）
- **固定递减位置** — 可选 DR 图标固定槽位不自动重排
- **拖拽位置修复** — 修复缩放不为1时拖拽位置计算错误
