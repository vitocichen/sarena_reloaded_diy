## 更新记录

### v1.0.1 (2025-04-03)

**自身递减模块 — 完全重写**

- 彻底重写自身递减（Self DR）模块，改为纯自身追踪，参考 MyDRs 插件设计
- 基于 `C_LossOfControl` API 实时监测玩家受到的控制效果
- 支持 7 种递减分类：昏迷、迷惑、瘫痪、定身、沉默、击退、缴械，可单独开关
- 递减状态文字显示（50% / IMM），直观了解当前递减阶段
- 可拖拽容器，位置自动保存
- 图标大小、间距、字体大小、增长方向（上/下/左/右）均可配置
- 倒计时旋转动画可单独开关
- 测试模式：图标上方显示功能名称，循环演示各递减状态，支持拖拽调整位置
- 修复 `/reload` 后残留旧框体导致图标重复显示的问题

**宠物框体修复**

- 修复竞技场中对手宠物框体不显示名字和血量的问题
- 当无法获取宠物信息时，回退显示本地化文字"宠物"

**界面与本地化**

- 自身递减配置面板全面汉化
- 移除旧的"自身+队友"追踪模式及相关配置项，简化设置界面

### v1.0.0

- **宠物框体** — 新增竞技场对手宠物生命条，支持位置/大小/颜色/文字等配置
- **右键焦点可配置** — 可关闭右键设焦点，兼容 Clique 等点击施法插件
- **DR 锚定模式** — 递减图标可选仅框体、仅血条、或双显
- **血条饰品镜像** — 可在血条旁显示饰品 CD
- **DR 免疫发光** — 递减免疫时图标脉冲发光
- **固定递减位置** — 可选 DR 图标固定槽位不自动重排
- **拖拽位置修复** — 修复缩放不为1时拖拽位置计算错误
